# README update for Discord

## Discord Bot Setup

1. Create a bot at https://discord.com/developers/applications
2. Copy token to .env
3. Invite bot to your server with `bot` and `applications.commands` scopes
4. Run: `pip install discord.py requests python-dotenv`
5. `python chaster_discord_bot.py`

Commands:
- `!status`
- `!addtime 30`
- `!freeze`
